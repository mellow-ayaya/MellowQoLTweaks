## Mellow QoL Tweaks

This addon is a collection of QoL additions that I wanted to implement in very specific ways to suit my weird tastes.

## What it do?

#### Sound alerts (enabled by default)
- Same concept as WoW's Audio -> Dialog -> Error Speech option
- Allows in depth customization for which errors to trigger sounds
- Each type of alert can be customized to a sound effect or voiceline
- Floating text that shows the name of the spell that failed
- Ignore list for spells you want to spam without getting blasted by sound alerts

#### Auto Mute (enabled by default)
- When you get automatically disconnected due to being AFK, it mutes the music so that the character selection screen doesn't endlessly play the theme song
- Restores previous music settings when logging back in
- Does nothing if music is already muted

#### Persistent UI Scale (disabled by default)
- Similar to all other addons that do this, except it does not only apply the scaling on login, it also applies it whenever the window is resized

#### Sounds and Fonts via shared media (mostly enabled by default)
- Friz Quadrata with cyrillic support in 5 tightness and boldness levels so that you can have one of most recognizable WoW fonts fit better in smaller spaces or appear bolder when needed
- Several sound packs and 2 voice packs (M&F) to be used with the Sound alerts feature
- 2 background textures :)
- All the fonts added by this addon can be disabled individually so media you don't plan to use doesn't clog your media selection menus
